# Avatars Directory

This directory stores archer profile photos/avatars.

## File Naming Convention
- `{timestamp}_{random}_thumb.{ext}` - 128x128px thumbnail
- `{timestamp}_{random}_full.{ext}` - 512x512px full size

## Supported Formats
- JPEG (.jpg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)

## Image Processing
- All images are automatically cropped to square (center crop)
- Two sizes generated: thumbnail (128x128) and full (512x512)
- Maximum upload size: 10MB
- Quality: JPEG/WebP at 90%, PNG at compression level 9

## Usage
- Thumbnail: Used in archer lists, scoring tables
- Full: Used in archer details, trading cards, results displays
