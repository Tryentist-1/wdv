## üéØ Feature 1: Status Flags Tracking System

### What It Does

Track five mandatory annual milestones for each archer to ensure eligibility and compliance. These status flags help coaches quickly see who has completed required items and who still needs to complete them.

### The Five Milestones

1. **Mandatory Safety Training** - Completion of required safety training
2. **Physical** - Current physical examination on file
3. **Athletic Clearance** - Athletic clearance documentation complete
4. **Equipment Checkout** - Equipment has been checked out to archer
5. **Tryout Attendance** - Archer attended required tryout sessions

### Key Features

**Status Tracking**
- One entry per archer per season/year
- Track completion status (complete/incomplete) for each milestone
- Record completion dates
- Add optional notes for each milestone

**Visual Dashboard**
- See team-wide compliance at a glance
- Color-coded indicators (green = complete, red = incomplete)
- Progress percentages for each milestone
- Quick view of which archers need attention

**Bulk Operations**
- Mark multiple archers as complete for a flag at once
- Efficient updates for large groups
- Save time on repetitive tasks

**Filtering & Search**
- Filter archers by completion status (e.g., "Show all archers missing Physical")
- Find specific archers quickly
- Export filtered lists for reports

**Email Integration**
- Filter archers by incomplete flags
- Generate email list automatically
- Open Gmail (or default email client) with pre-filled recipients
- Pre-written email templates for common reminders
- Privacy protection (recipients hidden in BCC)

### How Coaches Will Use It

**Daily Workflow:**
1. Open Status Flags section in Coach Tools
2. Select current season (e.g., "2024-2025")
3. View compliance dashboard to see overall team status
4. Filter by incomplete flags to see who needs attention
5. Update status flags as archers complete requirements
6. Use bulk operations to mark multiple archers at once

**Email Reminders:**
1. Filter archers missing a specific flag (e.g., "Physical")
2. Review the filtered list
3. Click "Email Selected Archers" or "Email All Filtered"
4. Email client opens with pre-filled recipients and message
5. Review and send reminder email

**Compliance Reports:**
1. View compliance dashboard for team overview
2. Export compliance report (CSV/PDF) for records
3. Share reports with administrators or parents

### Benefits

- **Time Savings:** Quick visual overview of team compliance
- **Better Communication:** Easy email reminders for missing requirements
- **Compliance Tracking:** Never miss a required milestone
- **Historical Records:** Track compliance across multiple seasons
- **Mobile-Friendly:** Works on phones (99% of usage)

## üéØ Feature 2: Enhanced Notes System

### What It Does

Improve the notes system with topics/categories, batch operations, formatted display, and dashboard widgets. This makes notes more organized, easier to find, and more useful for coaching.

### Key Features

**Topic-Based Organization**
- Categorize notes by topic (e.g., "Equipment", "Technique", "Goals", "Coach Comment", "Achievement")
- Create custom topics as needed
- Filter notes by topic for quick access
- Organize notes logically

**Global Notes for Batch Operations**
- Write a note once, apply to multiple archers
- Save note templates for reuse
- Efficient for common messages (e.g., equipment checkout notes)
- Apply to selected archers with one click

**Formatted Note Display**
- Support for Markdown formatting (bold, italic, lists, headers)
- Support for HTML formatting
- Rich text display in note history
- Better readability and organization

**Note History**
- View all notes for an archer in chronological order
- Filter by topic, date range, or search
- Formatted display (Markdown/HTML rendered properly)
- Easy navigation through note history

**Note Sharing**
- Share note content via SMS or email
- Filter archer list and send to selected archers
- Content automatically formatted for sharing
- Opens native SMS/Email apps on device

**Dashboard Embeds/Widgets**
- Display notes in dashboard widgets with different filters
- Create custom widgets (e.g., "Coach Comments" widget, "Achievements" widget)
- Each widget can have its own filter criteria
- Real-time updates as notes are created

### How Coaches Will Use It

**Creating Notes:**
1. Open archer detail or notes section
2. Select or create a topic (e.g., "Coach Comment", "Achievement")
3. Write note with optional formatting (Markdown or HTML)
4. Save note to archer's history
5. Note appears in note history and relevant dashboard widgets

**Batch Operations:**
1. Create a global note template (e.g., "Standard Equipment Package")
2. Select multiple archers
3. Apply template to all selected archers with one click
4. Each archer gets their own copy of the note

**Viewing Notes:**
1. Open archer's note history
2. Filter by topic (e.g., show only "Achievement" notes)
3. View formatted notes in chronological order
4. Search notes by content or topic

**Sharing Notes:**
1. View a note
2. Click "Share Note"
3. Select target archers (filter if needed)
4. Choose delivery method (SMS or Email)
5. Native SMS/Email app opens with pre-filled message
6. Review and send

**Dashboard Widgets:**
1. Create dashboard with multiple note widgets
2. Configure each widget with different filters:
   - "Coach Comments" widget: Shows all "Coach Comment" notes from last 30 days
   - "Achievements" widget: Shows all "Achievement" notes from this season
3. Widgets automatically update as new notes are created
4. View different note categories at a glance

### Suggested Topics

**Recommended Topics:**
- **Coach Comment** - General coaching feedback and observations
- **Achievement** - Milestones, personal bests, accomplishments
- **Equipment** - Equipment notes and checkout information
- **Technique** - Technique feedback and improvement areas
- **Goals** - Goal setting and progress tracking
- **Safety** - Safety-related notes and reminders
- **Attendance** - Attendance notes and patterns

**Custom Topics:**
- Coaches can create their own topics as needed
- Topics can be used for filtering and organization
- Topics help organize notes for different purposes

### Benefits

- **Better Organization:** Topics make notes easy to find and organize
- **Time Savings:** Batch operations save time on repetitive notes
- **Better Communication:** Formatted notes are easier to read and understand
- **Flexible Display:** Dashboard widgets show notes in different contexts
- **Easy Sharing:** Share notes via SMS or email with one click
- **Historical Tracking:** Complete note history for each archer
- **Mobile-Friendly:** Works on phones (99% of usage)

## üéØ Use Case Examples

### Use Case 1: Pre-Season Compliance Check

**Scenario:** Coach needs to ensure all archers have completed required items before season starts.

**Workflow:**
1. Open Status Flags section
2. Select current season
3. View compliance dashboard
4. See that 8 archers are missing Physical
5. Filter to show only archers missing Physical
6. Click "Email All Filtered"
7. Email client opens with pre-filled reminder message
8. Send email to all 8 archers
9. Update status flags as archers complete requirements

**Time Saved:** 15-20 minutes vs. manual email composition

### Use Case 2: Equipment Checkout Notes

**Scenario:** Coach needs to record equipment checkout for multiple archers.

**Workflow:**
1. Create global note template: "Standard Recurve Package - Checked out [DATE]"
2. Select all archers receiving equipment
3. Apply template to all selected archers
4. Each archer gets their own note with checkout date
5. Notes appear in "Equipment" topic filter
6. Easy to track who has checked out equipment

**Time Saved:** 10-15 minutes vs. individual note creation

### Use Case 3: Achievement Celebration Dashboard

**Scenario:** Coach wants to display recent achievements on a dashboard.

**Workflow:**
1. Create notes with topic "Achievement" as archers reach milestones
2. Create "Achievements" dashboard widget
3. Configure widget to show "Achievement" notes from this season
4. Widget automatically displays all achievement notes
5. View celebrates archer accomplishments
6. Widget updates automatically as new achievements are added

**Benefit:** Visual celebration of archer progress and milestones

### Use Case 4: Technique Feedback Sharing

**Scenario:** Coach wants to share technique feedback with archer via SMS.

**Workflow:**
1. Create note with topic "Technique" during practice
2. Note: "Focus on follow-through consistency. Anchor point is good."
3. Click "Share Note" on the note
4. Select archer from list
5. Choose "SMS" delivery method
6. SMS app opens with pre-filled message
7. Send message to archer

**Benefit:** Quick, direct communication with archers

## ‚ùì Questions for Review

### Status Flags

1. **Season Definition:** How should a "season" be defined? (School year? Calendar year? Custom range?)
2. **Auto-Reset:** Should status flags automatically reset at start of new season, or manual reset?
3. **Custom Flags:** Should coaches be able to add custom status flags beyond the 5 mandatory?
4. **Email Templates:** Should email templates be customizable per coach, or shared system-wide?
5. **School Email:** Where should school email address be configured? (Coach settings, config file, or manual entry?)

### Enhanced Notes

1. **Topic Management:** Who can create/edit topics? (Coaches only? Shared across coaches?)
2. **Note Privacy:** Should notes be private (coach-only) or visible to archers? (Future consideration)
3. **Embed Configuration:** Should embed configurations be saved per coach, or shared system-wide?
4. **Message Limits:** Should there be limits on SMS message length or number of recipients?
