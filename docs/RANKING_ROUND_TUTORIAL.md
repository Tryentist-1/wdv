## Overview

This tutorial will guide you through using the Ranking Round scoring application. The Ranking Round is used for competitive scoring with multiple archers on the same bale, typically 12 ends of 3 arrows each.

## 1. Setup

### Step 1: Set Your Bale Number

1. **Open the Ranking Round app** from the main menu
2. **Look at the top corner** of the setup screen
3. **Click on the Bale Number field** (defaults to "1")
4. **Enter your bale number** (1-99)
5. **Press Enter** or click outside to save

### Step 2: Select Your Bale Group Members

1. **Search for archers** using the search bar at the top
2. **Click the checkbox** next to each archer in your bale group
3. **Assign target letters** (A-H) to each archer:
   - Click the dropdown next to each selected archer
   - Choose a unique target letter (A, B, C, D, etc.)
4. **Verify all archers are selected** and have target assignments

### Step 3: Start Scoring

1. **Click the blue "Scoring" button** at the bottom
2. **You'll be taken to the scoring screen** showing "R Round - E1"
3. **Verify your bale number** is displayed correctly

## 2. How to Enter Scores

### Navigate Between Ends

- **Next End**: Click the green "E →" button to advance to the next end
- **Previous End**: Click the gray "← E" button to go back to the previous end
- **Current End**: The current end number is displayed in the header

### Enter Scores for Each Archer

1. **Click on any score cell** for an archer
2. **The keypad will appear** at the bottom of the screen
3. **Enter the score** using the keypad:
   - **X** = 10 points (inner gold ring)
   - **10** = 10 points (outer gold ring)
   - **9, 8, 7, 6, 5, 4, 3, 2, 1** = corresponding points
   - **M** = Miss (0 points)
4. **Click "➡️"** to move to the next arrow
5. **Repeat for all 3 arrows** per archer per end

### Score Entry Tips

- **Color coding**: Scores are color-coded (gold for 10/X, red for 8-9, blue for 6-7, etc.)
- **Auto-calculate**: Totals and running scores update automatically
- **Navigation**: Use the keypad arrows to move between cells
- **Clear**: Use the "Clear" button to reset a cell

## 3. How to Access Score Cards

### View Individual Archer Cards

1. **Click the "»" button** next to any archer's name
2. **The individual scorecard opens** showing:
   - Archer's full name and details
   - Complete score breakdown by end
   - Running totals and averages
   - X's and 10's count

### Navigate Between Archer Cards

- **Next Archer**: Click "Next →" to view the next archer's card
- **Previous Archer**: Click "← Prev" to view the previous archer's card
- **Back to Scoring**: Click "← Back" to return to the main scoring screen

### Card View Features

- **Complete score history** for all 12 ends
- **Running totals** and averages
- **X and 10 counts** for the round
- **Export options** (Save, Copy, SMS)

## 4. Save and Send

### Verify Scores

1. **Go to any archer's card** using the "»" button
2. **Review the complete scorecard** for accuracy
3. **Check running totals** and final scores

### Send Results via SMS

1. **Click "Verify & Send"** on any archer's card
2. **Review the bale totals** in the popup modal
3. **Click "Send SMS"** to open your phone's messaging app
4. **The formatted results** will be ready to paste into a Google Sheet

### Save Results

1. **Click "Save"** on any archer's card
2. **A screenshot** of the scorecard will be saved to your device
3. **Use for records** or sharing with coaches

### Copy Results

1. **Click "Copy"** on any archer's card
2. **Text format** of the results is copied to clipboard
3. **Paste anywhere** for sharing or record keeping

## Troubleshooting

### Common Issues

**Q: I can't find an archer in the list**
- A: Use the search bar to find archers by name
- A: Check if the archer needs to be added to the master list

**Q: Scores aren't saving**
- A: Make sure you click outside the cell or use the keypad navigation
- A: Check that you're connected to the internet (for initial load)

**Q: I need to change an archer's target assignment**
- A: Go back to setup using the "Setup" button in the footer
- A: Modify the target letter assignment and return to scoring

**Q: How do I start a new round?**
- A: Click "New R?" in the footer to reset and start fresh
- A: All current scores will be cleared

### Tips for Tournament Use

- **Test the app** before the tournament starts
- **Charge your device** fully before the event
- **Have a backup plan** (paper scoresheets) just in case
- **Practice the workflow** with sample data first
