‚úÖ PREFERRED PATTERN: Use flexbox container with input and button as siblings. Avoids absolute positioning issues.

Structure:

  • Outer div with flex, border, and focus-within:ring-2
  • Input with flex-1, bg-transparent, border-none
  • Button with fa-times-circle icon, toggle hidden class via JS

Why this pattern? Absolute positioning can fail due to parent transforms, z-index conflicts, or flex/grid contexts. Flexbox siblings are bulletproof.
